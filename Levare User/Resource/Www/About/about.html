This is a mobile application that reminds the end users about their appointments/tasks. The user, keys-in various reminders/alerts along with the date and time on which they wish to be reminded about , on the specified date and time the reminder pops up on their mobile screen with the link to the specific website where the payment(if any) has to be made, there are options where the user can snooze or cancel the reminder. Initially the user downloads the app for a free trail after which the user has to subscribe by making payment to enjoy the sophistication of this app.
